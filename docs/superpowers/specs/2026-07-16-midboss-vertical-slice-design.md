# 첫 중간 보스 수직 슬라이스 설계

## 목표

현재 연속 적 생성과 레벨업만 반복되는 런에 명확한 고저점을 만든다. 플레이 성과로 보스 진입을 앞당기되 런이 지나치게 짧거나 무한히 늘어지지 않게 하고, 보스 처치 뒤 강한 유물을 골라 다음 구간을 즉시 이어 간다.

이번 작업은 첫 중간 보스 1종, 보스 진입 진행도, 첫 유물 선택, 다음 구간 재개까지 포함한다. 두 번째 보스, 최종 보스, 스테이지 클리어, 메타 해금, 정식 그래픽·사운드는 포함하지 않는다.

## 보스 진입 진행도

생성된 적 수가 아니라 처치한 적의 위협 점수를 사용한다.

- 기본 적: `1점`
- 장갑 적: `2점`
- 사수 적: `2점`
- 첫 구간 목표: `70점`
- 최소 등장 시간: `120초`
- 최대 등장 시간: `210초`

`120초` 이후 목표 점수를 달성하면 보스 경고를 시작한다. 점수가 부족해도 `210초`에 도달하면 강제로 경고를 시작한다. 목표와 시간은 밸런스 설정 한곳에서 변경할 수 있어야 한다.

## 진행 상태

전투 진행 상태는 다음과 같다.

```text
running -> bossWarning -> boss -> bossRewardPaused -> running
running/bossWarning/boss -> levelUpPaused -> 직전 상태
any state -> defeated
```

- `bossWarning` 진입 즉시 일반 보충 대형 생성을 중단한다.
- 이미 화면에 있는 일반 적과 탄환은 유지한다.
- 약 `2초` 경고 후 화면 전환 없이 보스를 등장시킨다.
- 레벨업과 보스 진입 조건이 같은 프레임에 발생하면 레벨업을 먼저 처리한다.
- 패배는 모든 진행 상태보다 우선한다.
- 보스 처치 시 모든 적대 탄환, 낙하물, 공격 예고를 정리하고 유물 선택을 연다.
- 유물 선택 뒤 다음 구간 번호를 올리고 구간 시간, 보스 진행도, 일반 생성 시간을 초기화한다.
- 플레이어 체력, 일반 능력, 영구 구슬 상태, 선택한 보스 유물은 유지한다.
- 첫 수직 슬라이스는 첫 보스만 등장시킨다. 다음 구간은 연속 전투를 계속하지만 두 번째 보스 조건은 후속 작업 전까지 비활성화한다.
- 다음 구간은 위협 단계 1에서 시작하고 구간 60초 뒤 위협 단계 2가 된다. 적 하강 속도 `18px/s`는 유지한다.

## 보스 형태와 피해

첫 보스는 큰 사각 본체, 좌우 약점 2개, 중앙 코어로 구성한다.

- 좌우 약점: 각각 `14 HP`
- 중앙 코어: `36 HP`
- 총 실질 체력: `64`
- 목표 전투 시간: `45~75초`
- 본체 직접 적중: 피해 없이 구슬 반사
- 약점 하나 파괴: 공격 간격 소폭 감소
- 약점 둘 파괴: 중앙 코어 영구 노출
- 중앙 코어 파괴: 보스 처치
- 약점 판정은 정밀 조준 피로를 줄이도록 실제 표시보다 약간 넓게 둔다.
- 프로토타입 본체 크기는 약 `120x72px`, 보스 이동 속도는 `55px/s`로 시작한다.

직접 피해는 영구 구슬과 임시 구슬에 동일하게 적용한다. 폭발은 범위 안의 노출 부위 하나에만 한 번 적용해 약점 여러 개를 동시에 중복 타격하지 않는다.

## 보스 이동과 일반 적

보스는 화면 상단 전투 띠에서 천천히 좌우 이동한다. 일반 적과 실제 Arcade Physics 충돌을 사용하지 않고, 빈 공간을 계산해 이동 범위를 제한한다.

- 보스 이동 높이와 겹치는 일반 적의 좌우 구간에 안전 여백을 더해 장애물로 취급한다.
- 보스는 현재 자신이 속한 연결된 빈 구간 안에서만 왕복한다.
- 진행 방향이 막히면 감속하고 반전한다.
- 양쪽이 막히면 제자리에서 공격한다.
- 순간이동하거나 일반 적을 밀어내지 않는다.
- 적이 죽거나 아래로 내려가면 빈 구간이 합쳐지고 이동 범위가 자연스럽게 넓어진다.
- 불가피한 순간 겹침은 물리 반응 없이 통과시키고 일반 적을 보스보다 앞에 그린다.

보스 경고 뒤 일반 대형이 중단되므로 막힘은 영구적이지 않다. 첫 보스는 별도 일반 적 소환을 하지 않는다. 화면이 지나치게 비는지는 플레이테스트 후 판단한다.

## 공격 패턴

### 조준 사격

- 실제 보스 위치에서 발사한다.
- 플레이어 위치를 약 `0.6초` 예고한다.
- 예고 뒤 `1 피해` 탄환 3발을 부채꼴로 발사한다.
- 기존 일반 적 탄환 상한과 공유해 회피 불가능한 탄막을 방지한다.

### 수직 낙하 지원 사격

- 보스 위치와 무관한 화면 상단 지점에서 수직으로 떨어진다.
- 설정상 외부 지원 사격이나 소환 낙하물로 취급한다.
- 바닥 위치 2곳을 약 `0.8초` 예고한다.
- 직격 피해는 `2`다.
- 일부 표식은 현재 위치를 겨냥하고, 일부는 주변 이동 경로를 제한한다.

약점 단계에서는 두 패턴을 번갈아 사용한다. 약점 하나가 파괴되면 간격을 줄인다. 코어 단계에서는 제한적으로 두 패턴이 겹칠 수 있다. 모든 피해는 기존 `600ms` 무적 시간을 적용한다.

- 약점 2개 활성: 패턴 간격 `2.8초`
- 약점 1개 활성: 패턴 간격 `2.3초`
- 코어 노출: 패턴 간격 `1.9초`, 매 세 번째 공격에 두 패턴을 함께 예약

## 첫 보스 유물

보스 처치 시 전투를 완전히 정지하고 중복 없는 유물 3개를 보여준다. 첫 풀은 다음 4개다.

- `증설 탄창`: 영구 구슬 `+1`, 전체 최대 6개
- `회수 축전기`: 근접 직접 흡수한 구슬의 충전 횟수 `3 -> 5`
- `초동 증폭기`: 회수 후 첫 적중의 직접 피해 `+1`
- `연쇄 탄두`: 임시 분열 구슬도 폭발 효과를 상속

`연쇄 탄두`는 분열과 폭발 등급이 모두 1 이상일 때만 선택 풀에 들어간다. 현재 임시 구슬이 기본으로 폭발을 상속하는 동작은 제거하고 이 유물의 효과로 이동한다. 유효한 유물이 3개 미만이 되는 경우를 막기 위해 앞의 범용 유물 3개는 항상 선택 가능하다.

선택한 유물은 일반 능력 슬롯과 별도로 저장한다. 첫 수직 슬라이스에서는 유물 중복 획득을 허용하지 않는다.

## 구성 경계

- `EncounterDirector`: 구간, 처치 점수, 시간, 진행 상태, 보스 진입과 다음 구간 전이를 담당한다.
- `BossManager`: 보스 부위, 이동, 패턴 시계, 투사체, 피격과 처치를 담당한다.
- `BossMovementRules`: 일반 적 점유 구간에서 안전한 연결 이동 범위를 계산하는 순수 규칙이다.
- `BossRewardManager`: 유물 보유 상태, 선택 가능 조건, 효과 조회를 담당한다.
- `BossRewardOverlay`: 유물 3개 표시와 선택 입력만 담당한다.
- `CombatScene`: 각 시스템 event를 연결하고 pause와 표시를 동기화한다.

기존 `EnemyManager`, `OrbManager`, `TemporaryOrbManager`, `ProgressionManager`의 책임을 보스 로직으로 확장하지 않는다. 보스와 일반 적은 서로 다른 manager가 소유하되 직접 피해 event 형식은 가능한 한 공유한다.

## 실패와 경계 조건

- 최대 시간 도달 시 활성 일반 적 수와 관계없이 보스 경고를 시작한다.
- 경고 중 대기 중이던 보충 대형은 취소한다.
- 레벨업이 열려 있으면 경고 시간과 보스 공격 시계를 진행하지 않는다.
- 보스 처치와 일반 적 처치가 같은 프레임이면 피해·경험치를 정산한 뒤 보스 보상을 연다.
- 패배가 발생하면 경고, 보스 패턴, 유물 선택 대기를 모두 중단한다.
- 재시작은 구간, 보스 점수, 보스 유물, 보스 객체를 초기화한다.
- 구슬이 본체와 약점에 같은 물리 프레임에 겹쳐도 한 부위만 타격한다.

## 검증

단위 테스트:

- 처치 종류별 진행 점수와 `120초/70점`, `210초` 진입 조건
- 레벨업과 보스 조건 동시 발생 시 레벨업 우선
- 경고 이후 일반 대형 생성 중단과 pending 대형 취소
- 일반 적 점유 구간에 따른 보스 이동 제한, 반전, 정지, 공간 확장
- 약점 2개 파괴 전 코어 비활성, 이후 코어 활성과 처치
- 공격 예고, 단계별 간격, 탄환 상한, 피해값
- 폭발이 노출 부위 하나만 타격
- 유물 조건, 중복 없는 3개 선택, 효과 적용
- 보스 보상 뒤 다음 구간 초기화와 빌드·체력 유지

브라우저 자동 테스트:

- 처치 점수 조건과 최대 시간 조건에서 보스 경고가 시작된다.
- 경고와 보스 중 일반 보충 대형이 생성되지 않는다.
- 일반 적이 보스 이동을 제한하고 제거 후 이동 범위가 넓어진다.
- 약점 2개와 코어를 순서대로 파괴한다.
- 보스 보상 중 전투가 정지하고 선택 뒤 다음 구간이 재개된다.
- `연쇄 탄두`가 없을 때 임시 구슬은 폭발하지 않고, 획득 뒤에는 폭발한다.
- 패배와 재시작이 보스 상태와 유물을 초기화한다.

사용자 플레이:

- 보스가 평균 `45~75초` 안에 끝나는지
- 약점이 충분히 크고 반사 조준이 답답하지 않은지
- 일반 적 때문에 좁아진 이동 범위가 자연스럽게 보이는지
- 지원 사격 표식이 보스 위치와 분리돼도 공격 원인이 이해되는지
- 공격 겹침이 회피 불가능하게 느껴지지 않는지
- 유물 선택이 일반 레벨업보다 강하고 시각적으로 구분되는지

## 후속 작업

첫 보스 수직 슬라이스 플레이테스트 뒤 보스 소환 패턴, 추가 유물, 유물 연계 진화, 최종 보스, 스테이지 클리어와 메타 해금을 별도 설계한다.
