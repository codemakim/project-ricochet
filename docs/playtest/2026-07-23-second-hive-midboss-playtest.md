# 두 번째 군체 중간 보스 플레이테스트

## 실행

로컬:

```bash
npm run dev -- --host 127.0.0.1 --port 4173
```

브라우저에서 `http://127.0.0.1:4173`을 연다.

모바일 LAN:

```bash
npm run dev -- --host 0.0.0.0 --port 4173
```

노트북과 휴대폰을 같은 네트워크에 연결한 뒤 휴대폰에서
`http://<노트북-LAN-IPv4>:4173`을 연다. macOS Wi-Fi IPv4는 다음 명령으로 확인한다.

```bash
ipconfig getifaddr en0
```

## 확인 순서

1. 첫 보스를 처치하고 첫 보상을 고른 시점부터 시간을 잰다.
2. 위협 단계 3의 분열체 밀도와 벽 가장자리 파편을 확인한다.
3. 두 번째 경고 중 기존 적이 남고 새 증원이 생기지 않는지 본다.
4. 군체의 `4s` 보호막, `1.5s` 예고, `7s` 노출을 여러 번 경험한다.
5. 반사벽을 공격 경로에 이용하고 사격 모듈과 코어 탄막을 회피한다.
6. 모듈 우선과 코어 우선 공략을 각각 시도한다.
7. 두 번째 보상을 선택하고 세 번째 일반 구간이 즉시 이어지는지 확인한다.
